# フォルダ構成

- フォルダ構成は以下の通り

```
.
└── envs
    ├── config
    │   ├── dummy
    │   ├── id-pool-parameter.yml       ID-POOL スタック用パラメータファイル
    │   └── user-pool-parameter.yml     USER-POOL スタック用パラメータファイル
    ├── id-pool.yml                     ID-POOL スタック定義ファイル
    └── user-pool.yml                   USER-POOL スタック定義ファイル
```
